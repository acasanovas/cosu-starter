    selection

    @mixin selection($current-selector: false) { ... }

    Description

    Outputs the spec and prefixed versions of the ::selection pseudo-element.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $current-selector

    If set to true, it takes the current element into consideration.

    		Boolfalse

    Content

    This mixin allows extra content to be passed (through the @content directive).

    Example

    Usage

    .element {
  @include selection(true) {
    background-color: #ffbb52;
  }
}

    CSS Output

    .element::-moz-selection {
  background-color: #ffbb52;
}

.element::selection {
  background-color: #ffbb52;
}

    font-size

    @mixin font-size($font-size, $line-height: auto) { ... }

    Description

    Create a fully formed type style (sizing and vertical rhythm) by passing in a single value

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $font-size

    font-size in pixels

    		String—none
    $line-height

    line-height

    		Number or Inherit or Normalauto

    Example

    Usage

    .foo {
  @include font-size(12px);
}

    CSS Output

    .foo {
   font-size: 12px;
   font-size: 0.75rem;
   line-height: 2;
}

    Usage with line-height

    .foo {
  @include inuit-font-size(12px, 1.5);
 }

    CSS Output

    .foo {
  font-size: 12px;
  font-size: 0.75rem;
  line-height: 1.5;
}

    Usage with line-height none

    .foo {
  @include inuit-font-size(12px, none);
}

    CSS Output

    .foo {
  font-size: 12px;
  font-size: 0.75rem;
}

    Requires

    • [variable] font-size-default
    • [variable] line-height-default

    • https://github.com/inuitcss/tools.mixins/blob/master/_tools.mixins.scss
