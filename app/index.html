Web app boilerplate (WeBoil)

WeBoil is my kickstarter web app tool using Sass, Susy, Breakpoint and Normalize.css, designed with scalability and performance in mind.

Workflow

WeBoil workflow is automated and enhanced with Gulp, that offers a Sass documentation generator, CSS Autoprefixing, minified CSS output, Sass source maps, built-in preview server and synchronised browser testing.

Design

WeBoil is design-free and does its best to provide no styling of its own, so it can be used on any and all types of project without imposing a "look and feel".

Directory structure

WeBoil is focused on keeping things simple and obvious, and its approach happens to be quite similar to SMACSS.

Quick start

  1. Clone this repository.
  2. Download and install NodeJS.
  3. Install Bower and Gulp - npm install -g bower gulp.
  4. Install the required Node modules - npm install.
  5. Install the required packages - bower install.
  6. Ready, just run gulp.

Happy coding!
